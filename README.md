Library
============
[![GitHub Stars](https://img.shields.io/github/stars/MMarty11)](https://github.com/MMarty11/Library/stargazers) 
[![GitHub Issues](https://img.shields.io/github/issues/MMarty11)](https://github.com/MMarty11/Library/issues) 
[![Current Version](https://img.shields.io/badge/version-1.0.0-green.svg)](https://github.com/MMarty11/Library)

This is a library management system developed using WPF in C# with the main functions you might expect from a management system, such as managing existing books, managing registered users or a list of active loans.

## Features
- Modern Design
- Books management (add/edit/remove)
- Users management (add/edit/remove)
- Loans management (loan/return)
- Ability to search in DB
- Switch between light and dark mode
- Other awesome features yet to be implemented
